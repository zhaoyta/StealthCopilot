# 服务密钥配置

StealthCopilot 使用多个第三方服务来实现翻译、语音合成和画面同步功能。你需要分别注册这些服务并获取密钥。

**所有密钥均仅存储在你的本地设备上，不会上传到任何服务器。**

---

## 讯飞语音服务

用于将面试官的语音实时转写为文本，再通过机器翻译文本接口翻译为字幕；说话链也使用同一组文本翻译能力。

1. 前往[讯飞开放平台](https://www.xfyun.cn/)注册账号
2. 分别开通「实时语音转写 RTASR」和「机器翻译」服务
3. 在 RTASR 服务页面获取：
   - **RTASR App ID**
   - **RTASR API Key**
4. 在机器翻译服务页面获取：
   - **机器翻译 App ID**
   - **机器翻译 API Key**
   - **机器翻译 API Secret**

::: warning
讯飞不同接口的 API Key 可能不同。应用内也分成两组保存：RTASR 凭证只用于语音转写，机器翻译凭证只用于文本翻译。机器翻译会优先请求新版 `v1/its`，如果服务返回 `403 not found`，会自动降级请求控制台常见的 `v2/its`。
:::

[截图：讯飞控制台获取密钥位置]

## DeepSeek AI

用于分析面试问题的意图，并基于你的简历生成回答建议。

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/)注册账号
2. 在「API Keys」页面创建密钥
3. 复制 **API Key**（以 `sk-` 开头）

[截图：DeepSeek 控制台密钥页面]

## 讯飞声音复刻

用于训练你的专属音色，并在说话链中将翻译后的文本合成为真实人声。

1. 在讯飞开放平台为同一应用开通「一句话复刻（多风格版）」
2. 在服务页获取：
   - **声音复刻 App ID**
   - **声音复刻 API Key**
   - **声音复刻 API Secret**
3. 在应用「声音复刻」设置页获取讯飞训练文本，并按页面提示完整朗读录音
4. 点击「提交训练」后，应用会保存返回的 **Task ID**，表示训练任务已提交
5. 稍后点击「查询训练状态」；训练完成后应用会保存返回的 **Asset ID**
6. 保存 Asset ID 后，说话链可切换为你的克隆音色；如果不复刻或训练未完成，仍可使用默认音色

[截图：讯飞声音复刻 App ID / API Key / API Secret 位置]

::: tip
Task ID 和 Asset ID 都由应用在声音复刻流程中自动保存，不需要也不应该手动填写。Task ID 只代表训练已经提交，Asset ID 才代表声音复刻已经完成并可用于个人复刻音色；没有 Asset ID 时仍可使用默认音色。
:::

## 数字人视频输出（可选）

开启后，应用会在说话链输出时同步驱动一个数字人形象说话，面试官在摄像头里看到的是与语音口型对齐的数字人画面。**不配置时，说话链只输出虚拟麦克风音频，不影响其他功能。**

目前支持两种数字人服务：**Simli AI（推荐，个人可直接注册）** 和 **即构 ZEGO（企业级，需商务申请）**。

在「设置 → 设备」页面的「数字人视频输出」区块选择服务商后，填写对应配置即可。

---

## Simli AI 数字人（推荐）

Simli AI 支持个人开发者直接注册，无需商务申请。开启后音频仍由讯飞语音复刻直接输出到虚拟麦克风，Simli 负责生成与语音同步的视频画面。视频不会由 StealthCopilot 直接注册成系统摄像头，而是通过本机 OBS 浏览器源交给 OBS Virtual Camera。

### 第一步：注册 Simli AI 并创建数字人形象

1. 前往 Simli AI 官网注册账号
2. 在控制台创建一个数字人形象
3. 记录生成的 **API Key** 和该形象的 **Face ID**

[截图：Simli 控制台 API Key 和 Face ID 位置]

### 第二步：在应用内填写配置

在「设置 → API 密钥 → Simli AI」填写：
- **API Key**

在「设置 → 设备 → 数字人视频输出」区块：
- 选择服务商「Simli AI（推荐）」
- 填写 **Face ID**
- 勾选「启用数字人视频输出」

[截图：应用内 Simli 配置界面]

### 第三步：配置 OBS 输出

1. 打开 OBS App。
2. 新增「浏览器」来源。
3. URL 填写：

```text
http://127.0.0.1:18765/
```

4. 宽高建议先填 `512 x 512`。
5. 在 OBS 中点击「启动虚拟摄像机」。
6. 在会议软件中选择 `OBS Virtual Camera`。

::: tip 确保 OBS 已启动
StealthCopilot 只提供本机浏览器源，OBS App 必须保持打开并启动 OBS Virtual Camera。若会议软件看不到 OBS Virtual Camera，通常是 OBS 虚拟摄像机未启动，或 macOS 的 OBS Camera Extension 需要授权/重启。
:::

更多排障见[数字人与 OBS 配置](./digital-human)。

## 即构 ZEGO 数字人（企业级）

即构 ZEGO 数字人由云端同时生成音频和视频，面试官听到的和看到的均来自云端数字人。**须先联系即构商务申请 AIGC 数字人服务权限，不对个人开放。**

### 第一步：注册即构并开通 AIGC 数字人服务

1. 前往 [即构科技控制台](https://console.zego.im/) 注册账号
2. 创建一个应用，记录 **AppID**（纯数字）
3. 在「应用配置 → 服务器密钥」页面获取 **Server Secret**
4. 联系即构商务开通「AIGC 数字人」服务权限

[截图：即构控制台 AppID 和 Server Secret 位置]

### 第二步：创建数字人形象

1. 在控制台进入「AIGC → 数字人 → 形象管理」
2. 选择或上传一个数字人形象
3. 记录该形象的 **数字人 ID**

[截图：即构控制台数字人形象 ID 位置]

### 第三步：在应用内填写配置

在「设置 → API 密钥 → ZEGO 数字人（企业级）」填写：
- **AppID**
- **Server Secret**

在「设置 → 设备 → 数字人视频输出」区块：
- 选择服务商「即构 ZEGO（企业级）」
- 填写 **数字人 ID**
- 勾选「启用数字人视频输出」

[截图：应用内 ZEGO 数字人配置界面]

::: info CDN 转推拉流地址（可选，需要虚拟摄像头输出）
如果需要将数字人视频输出到虚拟摄像头，还需在即构控制台开启「混流转推 CDN」，并将 RTMP 拉流地址填入「CDN 转推拉流地址」字段。不填则只有音频输出，无视频。
:::

---

填写完成后，在应用设置的「服务密钥」标签页中输入以上信息，点击「保存」即可。

[截图：应用内密钥填写界面]
