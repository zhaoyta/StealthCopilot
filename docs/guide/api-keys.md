# 服务密钥配置

StealthCopilot 使用多个第三方服务来实现翻译、语音合成和画面同步功能。你需要分别注册这些服务并获取密钥。

**所有密钥均仅存储在你的本地设备上，不会上传到任何服务器。**

---

## 讯飞语音服务

用于将面试官的语音实时翻译为中文字幕，以及将你的中文语音翻译为英文。

1. 前往[讯飞开放平台](https://www.xfyun.cn/)注册账号
2. 创建应用，选择「实时语音翻译」服务
3. 在应用管理页面获取：
   - **App ID**
   - **API Key**
   - **API Secret**

[截图：讯飞控制台获取密钥位置]

---

## DeepSeek AI

用于分析面试问题的意图，并基于你的简历生成回答建议。

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/)注册账号
2. 在「API Keys」页面创建密钥
3. 复制 **API Key**（以 `sk-` 开头）

[截图：DeepSeek 控制台密钥页面]

---

## ElevenLabs 语音克隆

用于生成你的专属英文声音，让输出语音听起来自然真实。

1. 前往 [ElevenLabs](https://elevenlabs.io/) 注册账号（免费计划即可）
2. 在账号设置中找到 **API Key**
3. 完成声音克隆后，记录生成的 **Voice ID**

[截图：ElevenLabs API Key 和 Voice ID 位置]

::: tip
声音克隆可在应用内直接完成，无需在 ElevenLabs 网站手动操作。
:::

---

## Simli 画面同步

用于在视频通话中同步你的口型，让画面更加自然。

1. 前往 [Simli](https://simli.com/) 注册账号
2. 获取 **API Key**
3. 使用你的照片创建面部档案，记录 **Face ID**

[截图：Simli 控制台密钥和 Face ID 位置]

::: info 可选功能
如果不配置 Simli，应用将直接输出摄像头原始画面，其他功能不受影响。
:::

---

填写完成后，在应用设置的「服务密钥」标签页中输入以上信息，点击「保存」即可。

[截图：应用内密钥填写界面]
