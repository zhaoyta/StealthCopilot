# 面试历史记录

StealthCopilot 会自动保存每次面试的问答记录，帮助你回顾面试内容，同时让本次面试中的回答建议更加连贯。

## 历史记录的作用

面试是一个连续的对话过程，面试官可能会追问或引申之前的话题。开启历史记录后，系统在生成回答建议时，会自动参考本次面试中已经回答过的内容，避免前后矛盾或重复表述。

## 查看历史记录

在设置面板中，切换到**历史**标签页，可以看到按时间倒序排列的历史会话列表，每条记录显示使用的简历和问答轮数。

[截图：设置面板历史 Tab 列表界面]

点击任意一条记录，可以展开查看该次面试的完整问答内容。

[截图：历史记录详情展开界面]

## 删除历史记录

如需清除某次面试记录，在列表中找到对应条目，点击右侧**删除**按钮即可。删除操作不可恢复。

[截图：删除历史记录操作]

## 调整历史上下文范围

在设置面板**高级**标签页中，可以设置「历史轮数上限」，控制生成回答建议时参考最近几轮问答。默认值为 5 轮，设置为 0 则不注入历史上下文。

[截图：高级设置中的历史轮数上限配置]

## 数据隐私

所有面试历史记录**仅保存在本地设备**，不会上传至任何服务器。详见[隐私说明](./privacy)。
