# 快速开始

StealthCopilot 帮助你在视频面试中实时获得翻译字幕和回答建议，同时将你的中文回答转换为流利的英文语音输出。

## 系统要求

- **macOS** 12 及以上，或 **Windows** 10 及以上
- 摄像头 + 麦克风
- 稳定的网络连接

## 安装步骤

### 1. 下载安装包

前往 [Releases 页面](https://github.com/zhaoyta/stealthcopilot/releases) 下载最新版本：

- macOS：下载 `.dmg` 文件，拖拽到应用程序文件夹
- Windows：下载 `.msi` 安装包，按提示完成安装

### 2. 首次启动设置

首次运行时，系统将引导你完成以下设置步骤：

**步骤一：组件安装**

应用将自动检测并安装所需组件（虚拟声卡和虚拟摄像头），需要一次系统授权确认。

[截图：组件安装步骤界面]

**步骤二：填写服务密钥**

填写各服务的访问密钥，详见[密钥配置指南](./api-keys)。

[截图：服务密钥填写界面]

**步骤三：语言设置**

设置面试官使用的语言和你希望使用的语言。

[截图：语言设置界面]

**步骤四：声音复刻**

这一步可以跳过并使用默认音色。若想使用个人复刻音色，可按讯飞训练文本录制语音并提交训练；训练完成后，应用会保存 Asset ID。

[截图：声音复刻录制界面]

**步骤五：导入简历**

上传简历文件，系统将在本地建立索引以提供更精准的回答建议。简历文件不会上传到任何服务器。

[截图：简历导入界面]

### 3. 开始使用

设置完成后，在面试前：

1. 在会议软件（Zoom、Teams 等）中，将麦克风切换为 **StealthCopilot Virtual Mic**
2. 将摄像头切换为 **StealthCopilot Virtual Camera**
3. 点击应用中的「开始面试」按钮

[截图：主界面和管道启动状态]

面试中，幽灵提词窗（仅对你可见）将实时显示：
- 面试官说的内容（中文翻译）
- 基于你简历的回答建议

[截图：幽灵提词窗效果]
