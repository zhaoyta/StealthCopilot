# 隐私说明

## 哪些数据留在本地

以下数据**永远不会离开你的设备**：

- 你上传的简历文件
- 由简历建立的本地索引
- 所有服务密钥（存储在系统安全存储区）
- 面试过程中的对话记录

## 哪些数据会发送到第三方服务

为实现核心功能，部分音视频数据会实时传输到第三方服务：

| 数据类型 | 发送给谁 | 用途 |
|---------|---------|------|
| 面试官音频（实时） | 讯飞语音服务 | 实时翻译为中文字幕 |
| 你的中文语音（实时） | 讯飞语音服务 | 翻译为英文文本 |
| 面试问题文本 | DeepSeek AI | 分析问题意图，生成回答建议 |
| 目标语言文本 | TTS 服务 | 使用默认音色，或在你完成声音复刻后使用 Asset ID 生成你的克隆声音 |
| 你的视频画面（实时，可选） | Simli | 口型同步处理 |

以上第三方服务均有独立的隐私政策，请在使用前了解各服务的数据处理方式。

## 密钥安全

所有密钥存储在操作系统提供的安全存储区：
- **macOS**：钥匙串（Keychain）
- **Windows**：凭据管理器（Credential Manager）

密钥不会以明文形式保存在文件中，不会随截图或云同步泄露。
