# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅         |

## Reporting a Vulnerability

**请勿通过 GitHub Issues 公开报告安全漏洞。**

如果你发现了安全漏洞，请通过以下方式私下报告：

1. **推荐**：使用 GitHub [Private Vulnerability Reporting](https://github.com/zhaoyta/stealthcopilot/security/advisories/new)
2. **备用**：发送邮件至 zhaoyta@gmail.com，标题注明 `[SECURITY]`

我们承诺在收到报告后 48 小时内回复，并在修复后及时通知报告者。
